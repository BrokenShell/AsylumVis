# HRF Asylum Visualizer
## Database Visualization Example


### Tech Stack
- Logic: Python3.8
- Framework: Flask
- Templates: Jinja2
- Layouts: HTML5
- Styles: CSS 3
- Data: Pandas
- Graphs: Plotly
- Database: AWS RDS PostgreSQL
- Hosting: AWS Elastic Beanstalk
