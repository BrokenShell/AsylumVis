A Human Rights First Project

This is an example Flask app created to showcase the techniques used to parameterize plotly, pandas and SQL to create interactive graphs.

Enjoy, and be sure to checkout the code => AsylumVis GitHub Repo

WARNING! Be careful when parameterizing SQL! Don't open your data to SQL injection attacks by blindly executing raw user input. If this code is to be put in production you should take extra steps to ensure only clean SQL commands are executed.

Tech Stack

  • Logic: Python3.8
  • Framework: Flask
  • Templates: Jinja2
  • Layouts: HTML5
  • Styles: CSS 3
  • Data: Pandas
  • Graphs: Plotly
  • Database: AWS RDS PostgreSQL
  • Hosting: AWS Elastic Beanstalk
